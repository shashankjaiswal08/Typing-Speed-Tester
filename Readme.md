# Typing Speed Test

This is a simple web-based typing speed test application. It allows users to test their typing speed and accuracy by typing a given paragraph within a specified time limit.

## Features

- Randomly generated paragraphs for each test
- Timer to track the time taken for the test
- Word per minute (WPM) calculation based on the time taken and number of characters typed
- Accuracy calculation based on the number of correctly typed characters
- Start and reset buttons to begin or restart a test
- Display of test results, including WPM and accuracy

## Getting Started

1. Clone or download this repository.
2. Open the `index.html` file in a web browser.
3. Click the "Start Test" button to begin the test.
4. Type the paragraph displayed on the screen within the specified time limit.
5. The WPM and accuracy will be displayed after the test is completed.
6. Click the "Reset Test" button to restart the test.
